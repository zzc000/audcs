Installation Guide

  • Request trial/full license:

    Send an email to sales@anukj.com, including your business information and your Salesforce organization id to request trial/full license, package installation key

  • Install Managed Package:

    Install Salesforce managed package use the following link: https://login.salesforce.com/packaging/installPackage.apexp?p0=04t2v00000765ZR


  • Enter license in Salesforce organization:

    Go to "Setup -> Custom Code -> Custom Settings -> Manage [Ad Audience Secret] -> New [Default Organization Level Value] -> [Enter license]"


  • Assign permission set to users:

    Go to "Setup -> Users -> Permission Sets -> Advertising Audience User -> Manage Assignment"

User Guide

  • Create and authorise advertising account:

    Go to "APP [Advertising Audience] -> Advertising Account -> New"


  • Add leads and contacts to Campaign.
  • Create advertising audience:

    Go to "APP [Advertising Audience] -> Advertising Audience -> New"


  • Users will receive emails notification when salesforce start and finish uploading leads/contacts to advertising platforms.


  • Leads/Contacts available for use in advertising platform.
